# 我的仪表盘 v1.0.3

一个高端大气、功能丰富的个人仪表盘应用，采用深邃科技风格设计，支持多种主题切换。

## 🎨 设计理念

以"专业、高效、美观"为核心理念，打造一个集待办管理、工具箱、实时信息于一体的个人工作台。

## ✨ 核心功能

### 📋 待办事项管理
- 添加、完成、删除待办任务
- 优先级标记（高/中/低）- 采用彩色按钮组设计
- 本地存储持久化
- 实时统计（总数/已完成/剩余）

### 🌤️ 实时天气
- 和风天气API集成
- 显示温度、天气状况、湿度、风向风力
- 5秒超时保护机制
- 手动刷新功能

### ⏱️ 番茄钟
- 25分钟专注计时
- 5分钟休息计时
- 开始/暂停/重置控制
- 完成次数统计

### 🧮 计算器
- 完整四则运算
- 括号支持
- 实时显示表达式和结果

### ⏳ 倒计时
- 自定义时/分/秒设置
- 开始/暂停/重置控制
- 结束提醒

### 📝 快速笔记
- 添加笔记
- 查看历史笔记列表
- 删除笔记
- 本地存储持久化

### 📅 日历
- 显示当月日历
- 切换月份（上一月/下一月）
- 今日日期高亮显示

## 🎨 主题系统

支持5套精美主题，一键切换：

1. **深邃科技** (默认) - 深色科技风
2. **清新海洋** - 蓝色海洋调
3. **暗夜紫韵** - 紫色梦幻风
4. **暖阳金橙** - 橙色暖阳调
5. **森林墨绿** - 绿色自然风

主题自动保存，下次打开自动恢复。

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **样式**: CSS变量 + Flexbox/Grid布局
- **存储**: localStorage
- **API**: 和风天气API
- **部署**: GitHub Pages

## 📝 使用说明

### 待办事项
- 在输入框输入任务内容
- 点击"高"/"中"/"低"选择优先级
- 点击"添加"按钮或按回车键添加
- 点击复选框标记完成
- 点击🗑️删除任务

### 工具使用
- 点击左侧"工具箱"导航
- 点击任意工具卡片打开
- 使用完成后点击✕关闭

### 主题切换
- 点击顶部栏 🎨 按钮
- 选择喜欢的主题
- 自动保存并应用

## 🔒 数据存储

所有数据（待办、笔记、主题设置）均存储在浏览器本地(localStorage)，不会上传到服务器，保护您的隐私。
